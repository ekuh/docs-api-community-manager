When you login to API Community Manager as an administrator, you see three tabs, *Manage API Publication*, *Manage Communities*, and *Manage Community Users*.

*Manage API Publication* shows the list of APIs and the interface to [Publish APIs].

*Manage Communities* shows:

* Builder - [preview community], branding and customization
* Workspace - access to [Salesforce Community Cloud Workspace], which offers all the functionality of your community, including [moderate forums], engagement dashboards, and configuring advanced features.
* Content Management - [publish content]
* Open community - [view the live community site]

*Manage Community Users* shows the ability to control approval and activation for the community's user accounts.
