To show a live preview of the community's site, including APIs in *Preview Mode*, open the ACM administrator interface and select *Community Builder*.

image::builder.png[Site Builder]

Select *Preview*.

To return to the site builder, select *Back to Builder*.

To view the live site, open the ACM administrator interface and select *Open Community*.
