To see a live preview of your community's site, including APIs only visibile to the administrators, open the ACM Control Panel and select *Community Builder*.

image::builder.png[Site Builder]

Select *Preview*.

To return to the builder, select *Back to Builder*.

Use *Community Builder* to *Publish* the community.

Open the ACM Control Panel, select *Community Workspace*, and make the community *Active*.

To view the live site when the community is published and active, open the ACM Control Panel and select *Open Community*.
