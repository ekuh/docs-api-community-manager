Administering your API Community requires performing basic maintenance of your ACM Salesforce organization, monitoring important notifications about new ACM package releases and downtime, upgrading Anypoint API Community Manager, and if necessary, getting support from MuleSoft.

These pages explain how to perform each action:

* Salesforce documentation: https://help.salesforce.com/articleView?id=setup_overview.htm&type=5[Maintain an Organization, role=external, window=blank]
* xref:notifications.adoc[Monitor Important Notifications]
* xref:update-acm.adoc[Update API Community Manager]
* xref:mulesoft-support.adoc[Contact MuleSoft Support]
